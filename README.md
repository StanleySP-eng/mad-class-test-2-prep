# MAD Class Test 2 Prep — Navigation, Plug-ins & Bluetooth

Study materials for **Mobile Application Development (ET0893)**, Class Test 2 — covering **Chapters 5, 6 and 7**.

Work through the study guide, then test yourself with the practice questions. The questions use **different wording, code and values** from the test — they drill the same *ideas*, so you're building understanding, not memorising answers.

## Contents

| File | Purpose |
|---|---|
| [`STUDY_GUIDE.md`](STUDY_GUIDE.md) | Concept-by-concept walkthrough (by chapter) + cheat sheet + night-before recap |
| [`QUESTIONS.md`](QUESTIONS.md) | 30 practice questions with collapsible answers, code snippets and explanations |

## How to use

1. **Read `STUDY_GUIDE.md`** first — understand the concepts.
2. **Try `QUESTIONS.md`** closed-book — the `<details>` tags hide answers until you click.
3. **Skim Part 3** of the study guide (night-before recap) right before the test.

## Topics covered

**Chapter 5 — More Widgets & Basic Navigation**
- Routes, the `Navigator` widget, `push` / `pop`, `SystemNavigator.pop()`
- `BuildContext` and `MaterialPageRoute`; `ListView` vs `ListView.builder`; `GestureDetector`

**Chapter 6 — Flutter Plug-ins & More Navigation**
- API vs plug-in; Dart API vs Flutter API; Platform Channels
- `webview_flutter`; the `..` cascade operator; `google_maps_flutter`
- Passing data during navigation

**Chapter 7 — Microcontroller Communication**
- Bluetooth basics and profiles (SPP / RFComm / GATT); HC-05 / HC-06 + Arduino; pairing
- The Android Bluetooth stack; Flutter → native via Platform Channels; AT commands
